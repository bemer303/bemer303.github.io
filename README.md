# About
Hi, I’m Bryan Emer, a Full Stack Engineer transitioning into AI/ML and Applied Data Science with a focus on machine learning, data analysis, and scalable data systems. My work combines technical implementation with problem-solving in areas like predictive modeling, data storytelling, and end-to-end data pipelines.

This portfolio highlights projects that reflect both analytical depth and practical engineering. Recent work includes a diabetes risk prediction model using logistic regression, a childcare affordability analysis built around data storytelling and visualization, and a distributed Spotify popularity prediction pipeline using NiFi, HDFS, Hive, Spark MLlib, and HBase.

I’m especially interested in roles where I can apply machine learning and data engineering to real-world business problems. My goal is to build systems and analyses that are not only technically sound, but also useful, interpretable, and clearly communicated.
